# Ana Clara & Isaque - Wedding Countdown

Countdown elegante para o casamento de Ana Clara & Isaque.

**Data:** 15 de Outubro de 2026
**Local:** Castelo de Itaipava

## Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS (CDN)

## Executar localmente

```bash
npm install
npm run dev
```

## Build para produção

```bash
npm run build
```

O build será gerado na pasta `dist/`.

## Deploy

Compatível com:
- Vercel
- Lovable
- Netlify
- GitHub Pages
